jenis-jenis event luau

Event biasanya merujuk pada RBXScriptSignal. ini adalah cara sebuah objek memberi tahu script bahwa "sesuatu" telah terjadi.

  1. Players (Service)

    • PlayerAdded: Menjalankan fungsi saat pemain join ke server
    • PlayerRemoving: Menjalankan fungsi saat pemain keluar dari server

  2. Player (Object)

    • CharacterAppearanceLoaded: Jalan saa seluaruh aset karakter (Baju, Aksesoris) selesai dimuat.
    • CharacterAdded: Jalan saat model karakter masuk ke workspace
    • CharacterRemoving: Jalan saat karakter dihapus (Misal: saat mati atau ganti karakter)
    • Chatted: Menangkap pesan teks yg diketik pemain di chat

  3. Humanoid

    • Died: Mendeteksi saat HP menjedi 0
    • HealthChanged: Jalan setiap kali HP bertambah atau berkurang
    • SatateChanged: Mendeteksi perubahan status (Misal: dari lari ke melompat / berenang)
    • Climbing/FallingDown/Jumping/Running/Swimming: Event spesifik
